Submitting Problems

You're welcome to submit problems for inclusion in the Stereotopicity Trainer. Just drop me a line at mevans@metallacycle.com and include the following information for each problem:

  • A ChemDraw (CDX) structure of the file with a common or IUPAC name, drawn using this template. Don't forget to highlight the groups of interest in orange.
  • A tab-delimited line of text with this form: compound-name    correct-relationship    feedback-when-incorrect    feedback-when-correct
  • A .jmol file containing the structure with the groups of interested selected. Build this using Jmol's model kit mode or Jmol@NetMol.
Problems submitted by others will be posted with attribution.
